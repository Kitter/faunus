Version Compatibilities

The Faunus project is growing along with the rest of the Aurelius Graph Cluster technology. Below are version compatibilities between the various versions of Faunus and Titan. In order to reduce confusion, Faunus has skipped the 0.2.0 version in order to sync with Titan’s versioning.

Faunus Titan Hadoop
0.1-alpha 0.1.0 1.0.y
0.1.0 0.2.0 1.0/1.y
0.1.1 0.2.1 1.0/1.y
0.3.0 0.3.0 1.0/1.y
0.3.1 0.3.1 1.0/1.y
0.3.2 0.3.2 1.0/1.y
0.4.0 0.4.0 1.0/1.y
0.4.1 0.4.1 1.0/1.y
0.4.2 0.4.2 1.0/1.y
0.4.3 0.4.3 1.0/1.y
0.4.4 0.4.4 1.0/1.y
  • Faunus 0.1-alpha’s SequenceFile format is different than Faunus 0.1.0’s SequenceFile format due to the introduction of a variable-width encoding optimization.
